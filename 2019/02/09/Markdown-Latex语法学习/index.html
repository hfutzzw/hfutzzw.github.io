Markdown&&Latex语法学习

Type
  • Latex

工欲善其事，必先利其器。对于经常需要写各种科技论文（英文/中文）以及技术博客的童鞋而言，掌握常用的MarkDown、Latex语法尤为重要。这篇博客讲述一些最频繁使用的MarkDown、Latex语法以及自身所填的一些坑。

从Word走来

我们最开始接触的排版软件是微软出品的Office,它是一个足够强大的软件，但当涉及大量的数学公式、图表的编号和交叉引用时，Office内置的丑陋公式字体和薄弱的交叉应用功能就显得苍白无力了。
但大部分学校的本科毕业论文好像并不支持Latex, 所以还是有必要扯一下Office。

Word

对于word而言，MathType是支持最好的公式编辑软件，但是MathType是收费的，而且价格昂贵。这里提供一个下载链接, 提取码是:7xin。安装好并破解就可以永久使用了。

PPT

对于ppt而言，有一个比较好的开源插件支持Latex语法的使用:IguanaTex。它会调用Latex编译器来编译Latex代码并生成无损的图片(意味着不必担心放大十倍而模糊)甚至矢量图。它支持所有的Latex语法，所以我们可以用这个插件键入代码生成数学公式、流程图等并以ppt的形式展示。

走到Web端去

在本地书写markdown语言并渲染成html是写技术博客的常用做法。
书写markdown语言推荐使用VScode及其提供的MarkDown在本地实时预览的插件：Markdown Preview Enhanced。顺带说一下，开源的VScode真是神器，它在github上的star非常高。
